CLI for Image Processing (C/C++)

  • Personal project for the CS 3258 Intro to Computer Graphics course.
  • An OpenGL based image processing CLI built from scratch. Its functionalities include:
    • Reading and batch executing commands in UTF-8 files.
    • Reading, parsing and displaying TIFF binary files.
    • Manipulating images through discrete convolution filters (Gaussian, Lanczos, Mitchell, Hann, Hamming, sinc, box, tent, etc.).
    • Drawing geometric shapes on black canvas and changing their 3D transformations via matrix.
    • Rendering ray-traced images of 3D geometric shapes with different color & texture properties. Above is the image rendered by the program.

NaturalScene (Unity C#)

  • Personal research project designed for assessing human distance estimation abilities when wearing Microsoft HoloLens.
  • Project consists of:
    • a large naturalistic virtual terrain;
    • 6 different objects that appear at 5 different distances away from subject in random order on voice command;
    • floating text UI to indicate experiment data and status.
  • Project was supplied to the University of Utah Department of Psychology for testing.
  • Testing results were submitted to IEEE VR 2020 conference and published to IEEE XPlore, titled The Role of Viewing Distance and Feedback on Affordance Judgments in Augmented Reality.

NarrowingWalls (Unity C#)

  • Personal research project designed for assessing human affordance perception abilities when wearing Microsoft HoloLens.
  • Project consists of:
    • two giant virtual walls with a gap in between that change gap width on voice command or subject movement;
    • interactive buttons to change subject shoulder width;
    • floating text UI to indicate experiment data and status.
  • Project was supplied to the University of Utah Department of Psychology for testing.
  • Testing results were submitted to IEEE VR 2020 conference and published to IEEE XPlore, titled The Role of Viewing Distance and Feedback on Affordance Judgments in Augmented Reality.

Digital Cultural Heritage
(Unity C#, Python, PyTorch)

  • Ongoing personal research project for Oculus VR.
  • Aimed to raise public awareness and aid researches on historic stereoscopic images primarily made in mid-19th and 20th centuries.
  • Automates the process of segmenting the images for VR with pre-trained neural networks.
  • A collection of 150 hand-cropped images was created as training database to train a faster R-CNN detection model with Detectron2.
  • General workflow of the program:
    • Fetch hi-res stereo images from the New York Public Library database via its public API, store them locally along with metadata.
    • Segment those images with the detection model into left and right components, which are automatically imported to Unity.
    • Unity exhibits the collection with features to browse and display the images in stereo mode. Users are allowed to make copies and freely adjust their locations in the virtual room.
  • Program is planned to be first released to history researchers in the field. An Oculus Store / Steam VR version of this project is due for public release in the near future.

HoloScene (Unity)

  • Ongoing Unity project to record or livestream interactive multimedia sessions in VR and make the sessions viewable and interactable on any mobile device as animated 3D objects.
  • Project consists of two parts:
    • A Windows desktop recorder that records or livestreams sessions using VR headsets and upload the bundled sessions to an AWS server.
    • A mobile (iOS, Android) client that streams recorded or live sessions from the server.
  • My primary focus is to build & develop the mobile client application with more features as well as maintaining the server & CDN for content distribution.
